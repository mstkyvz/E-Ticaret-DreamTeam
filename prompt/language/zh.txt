描述性简短句子: {与产品相关的简洁描述}

指示：

使用上述“描述性简短句子”，按照以下JSON结构生成输出：

{ "product_name": "{产品名称，如果有品牌且在图片中注明品牌，则包括品牌}", "product_detailed_description": "{根据图片和描述性句子得出的丰富的产品详细描述}" }

规则：

只返回JSON结构。不要添加任何其他文本或解释。
如果图片中品牌可见，但“描述性简短句子”中未注明，请将品牌名称添加到“product_name”字段中。
除图片和“描述性简短句子”外，不要进行任何评论或假设。
如果产品的数量（克、升等）可以从“描述性简短句子”或图片中获取，请将其包含在“product_name”字段中。
在“product_detailed_description”字段中，突出产品的优点、用途以及任何特别的属性（例如，有机、手工制作）。
JSON结构中的键应该是英文的。
JSON键名应保持为"product_name" 和 "product_detailed_description"。
示例：

输入：

描述性简短句子: 250克冷榨橄榄油。

输出：

{ "product_name": "冷榨橄榄油 - 250g", "product_detailed_description": "这250克冷榨橄榄油是由优质橄榄制成，富含健康的脂肪酸，适合为您的菜肴增添丰富的风味。" }

输入：

描述性简短句子: 1升手工石榴浓缩汁，天然。

输出：

{ "product_name": "手工石榴浓缩汁 - 1L", "product_detailed_description": "1升手工制作的石榴浓缩汁，使用新鲜石榴汁制作。非常适合为沙拉和菜肴增添风味。无添加剂。" }
