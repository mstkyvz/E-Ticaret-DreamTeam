Танымляющее Краткое Предложение: {краткое и точное описание продукта}

## Инструкции:

Используя указанное выше "Танымляющее Краткое Предложение", создайте вывод, следующий структуре JSON ниже:

{
  "product_name": "{название продукта, если на изображении указана марка, добавьте её также}",
  "product_detailed_description": "{детальное описание продукта, обогащенное информацией, полученной из изображения и краткого описания}"
}

## Правила:

1. **Выводить только структуру JSON**. Не добавлять никаких дополнительных текстов или пояснений.
2. Если на изображении указана **марка**, а в "Танымляющее Краткое Предложение" она не указана, добавьте её в поле "product_name".
3. **Не делайте предположений или интерпретаций** за пределами изображения и "Танымляющее Краткое Предложение".
4. Включите **количество** продукта (граммы, литры и т.д.) из "Танымляющее Краткое Предложение" или изображения в поле "product_name".
5. В поле "product_detailed_description" выделите преимущества, области применения и, если есть, особые характеристики продукта (например, органический, ручной работы).
6. Ключи структуры JSON должны быть на английском языке.
7. Ключи JSON должны быть "product_name" и "product_detailed_description". Не изменяйте их.

## Примеры:

**Вход:**

Танымляющее Краткое Предложение: 250 грамм оливкового масла, холодного отжима.

**Выход:**

{
  "product_name": "Оливковое Масло Холодного Отжима - 250г",
  "product_detailed_description": "Это оливковое масло холодного отжима весом 250 грамм произведено из высококачественных оливок. Содержит полезные жирные кислоты и идеально подходит для добавления насыщенного вкуса в ваши блюда."
}

**Вход:**

Танымляющее Краткое Предложение: 1 литр ручной работы гранатового соуса, натуральный.

**Выход:**

{
  "product_name": "Гранатовый Соус Ручной Работы - 1л",
  "product_detailed_description": "Этот гранатовый соус ручной работы объёмом 1 литр сделан из свежевыжатого сока гранатов. Идеально подходит для добавления вкуса в салаты и блюда. Не содержит добавок."
}
